# Fyntrum — Corporate Financial Analyzer

Herramienta interna de análisis financiero para PYMEs ecuatorianas.

## Servicios
- Viabilidad Crediticia (desde $250)
- Consultoría Financiera Empresarial (desde $350)
- CFO Externo Mensual ($500/mes)

## Uso
Abre `index.html` en el navegador o accede al dominio de Vercel.

Ingresa tu **Anthropic API Key** en el sidebar para activar el Asistente IA y el generador de informes.

## Stack
- HTML / CSS / JavaScript puro — sin dependencias de build
- Chart.js (CDN) para gráficos
- Anthropic API (Haiku 4.5) para análisis IA
- Tabler Icons (CDN) para iconografía

## Deploy en Vercel
1. Sube este repositorio a GitHub
2. Importa el repo en [vercel.com](https://vercel.com)
3. Vercel detecta automáticamente el sitio estático
4. Deploy en un clic
